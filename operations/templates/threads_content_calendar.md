## 月曜投稿：「問いかけ型」（コメント誘発）

```
【テンプレートA：整体院・美容室オーナー向け】

「お客さんが来ない日と来すぎる日、どっちがしんどいですか？」

集客コンサルとかFPに相談するほどじゃないけど、
正直どっちがキツいかって聞かれたら、
意外とみんな「来ない日」より「来すぎる日」が怖いって言うんですよね。

キャパ超えて疲弊して、また客減って——という繰り返し。

自営業のお金と精神力のバランス、
インタビューで聞かせてもらってる内容が毎回深くて刺さります。

あなたはどっちがしんどいですか？コメントで聞かせてください。
```

```
【テンプレートB：ネイリスト・個人サロン向け】

「独立して一番後悔したこと、正直に教えてもらえますか？」

取材してきた中で「もっと早く知りたかった」って声が多いのが
"お金の話"なんです。

確定申告・保険・老後——
「開業したら自分でなんとかしなきゃ」ってわかってるけど
誰に聞けばいいかわからなかった、って。

独立3年以内の方、「後悔ポイント」コメントで共有してもらえると
同じ悩みを持つ人の参考になります💬
```

## 水曜投稿：「事例・ストーリー型」（信頼構築）

```

「月商80万の整体院オーナーが
『一番の経営リスクは自分が倒れること』と言っていた話」

先月インタビューした方の言葉が刺さりました。

年間売上は安定してる。予約も埋まってる。
でも——自分が1週間寝込んだら、即収入ゼロ。

「もし交通事故にあったら？」
「子どもの学費が重なった時期に廃業したら？」

保険でカバーできてる部分、
実は全然できてない部分があった、と。

個人事業主のリスク設計、
考えたことありますか？
```

```

意外と身近にあります。
```

## 金曜投稿：「参加募集型」（CTA直球）

```
【テンプレートA：フレンドリー募集】

【インタビュー参加者を探しています🙋】

✅ 整体院・美容室・ネイルサロンを経営している
✅ 個人〜少人数スタッフ（5名以下）
✅ お金・経営・将来のことを少し話してみたい

こんな方に、30〜40分のオンラインインタビューをお願いしています。

謝礼あり。営業なし。
（取材後にFPのご紹介はご案内することがありますが、
強制一切なし・無料相談のみ）

「なんか面白そう」くらいの気持ちで大丈夫です。
DMか「参加したい」コメントください😊
```

```
【テンプレートB：限定感演出】

「今月あと3枠だけ受け付けます」

ヒトモノチイキが行っているサロンオーナーインタビュー。

▶ 経営者としての本音が聞きたい
▶ 個人事業主のリアルなお金事情を取材したい

という目的でやっています。

毎月10名限定で実施していますが、残り3枠になりました。

・整体院 / 美容室 / ネイルサロン経営者の方
・フリーランス・個人事業主の方

「どんなことを聞かれるの？」という質問もDMで気軽にどうぞ📩
```
