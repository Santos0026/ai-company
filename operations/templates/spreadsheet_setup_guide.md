## ■ シートB：BO管理・紹介トラッキングシート

### ▶ ヘッダー行（コピペ用）

```
紹介先名	紹介元（掲載者名）	連絡先	初回連絡日	アポ確定日	インタビュー完了	FP申込	ステータス	担当BO	備考
```

> ※ シートAとは別タブで管理。シートAの「紹介先」欄から転記して使用します。

---

### ▶ 各列の入力ルール

| 列名 | 入力形式 | 入力例 | 必須 |
|------|----------|--------|------|
| 紹介先名 | 名前（敬称なし） | 山田太郎 | ✅ |
| 紹介元（掲載者名） | シートAの掲載者名と統一 | 花屋みどり | ✅ |
| 連絡先 | LINE ID / 電話番号 / Instagram ID | @yamada_t | ✅ |
| 初回連絡日 | YYYY/MM/DD | 2026/04/20 | ✅ |
| アポ確定日 | YYYY/MM/DD（未確定は空欄） | 2026/04/25 | 任意 |
| インタビュー完了 | ○ / ✗ / 対象外 | ○ | ✅ |
| FP申込 | ○ / ✗ / 検討中 | 検討中 | ✅ |
| ステータス | 未連絡 / 連絡済 / アポ確定 / 完了 / 離脱 | 連絡済 | ✅ |
| 担当BO | 担当者名 | Santos | ✅ |
| 備考 | 引継ぎ・特記事項 | 来週末希望と返信あり | 任意 |

---

### ▶ ステータスの使い方

```
未連絡 → 連絡済 → アポ確定 → 完了
                        ↓
                       離脱（反応なし・辞退）
```

- **完了**：インタビューまたはFP申込が完了した状態
- **離脱**：2週間以上反応なし、または明確に断られた場合
