## シートA：インタビュアー入力シート

### ヘッダー行（コピペ用・Googleスプレッドシートの1行目に貼り付け）

```
記録日	インタビュアー名	掲載者名	掲載者Instagram	選んだアンケート	相談テーマ	FP日程ステータス	紹介先①名前	紹介先①連絡先	紹介先②名前	紹介先②連絡先	備考
```

### 各列の入力ルール

| 列 | 項目 | 入力ルール |
|----|------|-----------|
| 記録日 | インタビュー実施日 | YYYY/MM/DD形式（例：2026/04/17） |
| インタビュアー名 | 担当者名 | 略称OK（例：田中・藤堂） |
| 掲載者名 | 店舗名または屋号 | 正式名称で記入 |
| 掲載者Instagram | @アカウント名 | @なしで入力（例：yamada_nail） |
| 選んだアンケート | FP・クレカ・SNS等 | 選択したアンケート名を記入 |
| 相談テーマ | FP面談の相談内容 | 例：老後資金・現在の保険見直し・教育費 |
| FP日程ステータス | 未定・調整中・確定済 | 確定の場合は日時も記入 |
| 紹介先①名前 | 紹介してもらった方の名前 | フルネームでなくてもOK |
| 紹介先①連絡先 | 電話・LINE等 | 不明な場合は「掲載者から連絡予定」と記入 |
| 紹介先②名前 | 同上 | |
| 紹介先②連絡先 | 同上 | |
| 備考 | 特記事項 | 温度感・懸念点など自由記入 |

### インタビュアーへのLINE共有文面

```
【お願い】インタビュー情報の記録について

インタビューが完了したら、当日中に以下のシートに入力をお願いします！

▼ 記録シート（シートA）
[GoogleスプレッドシートのURLをここに貼り付け]

入力する内容：
・掲載者のお名前・Instagram
・選んでいただいたアンケート（FP等）
・相談テーマ（FP面談で話す内容）
・FPの日程ステータス
・ご紹介いただいた方の情報（いれば）

⚠️ 特に「相談テーマ」と「FP日程ステータス」は
バックオフィスが後追い電話の前に必ず確認します。
当日中の入力をお願いします！

個人情報なので社外への共有・転送は禁止です。
よろしくお願いします😊
```

## シートB：紹介トラッキングシート（BO管理）

### ヘッダー行（コピペ用）

```
紹介先名	紹介元（掲載者名）	連絡先	初回連絡日	アポ確定日	インタビュー完了	FP申込	ステータス	担当BO	備考
```

### ステータスの流れ

```
未連絡 → 連絡済（反応待ち）→ アポ確定 → インタビュー完了 → FP申込完了
                                                          ↓
                                                       離脱（理由を備考に）
```

### BOの運用ルール

1. シートAに紹介先情報が入ったら、BOが当日中にシートBに転記
2. 3営業日以内に初回連絡を実施
3. 毎週月曜にステータスを全件確認・更新

---

## 注意事項

- 両シートとも社外秘（スプレッドシートの共有範囲を「リンクを知っている人」ではなく「特定のユーザーのみ」に設定すること）
- 記入ミス・空欄があった場合はBOがインタビュアーに当日中に確認
