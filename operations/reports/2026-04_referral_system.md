# 紹介獲得の組織化設計

**作成**: 2026-04-17 by Ryo（リサーチ担当）
**目標**: 月+20件のアポを紹介経由で獲得する

---

## 改訂版紹介スクリプト（全文）

現行の問題点：
- 「お知り合いいますか？」（Yes/No質問）→ 思考が止まる
- 自社都合フレーム → 相手に動機が生まれない
- 紹介のリターンがない

**設計原則：「あなたが友人に贈り物をする」フレームで設計**

---

### 【ステップ1】インタビューの余韻を使う

「今日は本当にありがとうございました。〇〇さんのお話、すごく刺さりましたし、読んでくれた方に響くと思います。掲載後の反応ぜひ教えてください！」

*(数秒間を置く)*

---

### 【ステップ2】紹介を「お願い」でなく「お裾分け」としてフレーミング

「一つだけご相談してもいいですか。

〇〇さんの周りで、同じように地道にお仕事されていて、でもあまり表に出ていないな…という方、いらっしゃいますか？

取材の話をすると、みなさん最初は『自分なんかが…』って遠慮されるんですよ。でも掲載後に『やってよかった』とおっしゃる方がほぼ全員で。〇〇さんがその方に『掲載してみない？』って声をかけてくださるだけで、その方にとってはすごく背中を押してもらえることになると思うんです。」

---

### 【ステップ3】「2名」を具体的に引き出す ← ここが核心

「業種は問わないので、パッと思い浮かぶ方、**2名だけ**教えていただけますか？同業者でも、全然違う業種でも大丈夫です。

*(メモを広げて書く姿勢を見せる)*

お名前と、どんなお仕事の方か、だけで大丈夫です。」

---

### 【ステップ4】連絡先を獲得する

「ありがとうございます！もし連絡先をご存知でしたら、LINEや電話番号を教えていただくか、難しければ『〇〇さんから紹介してもらった』とお伝えしてから弊社から連絡してもいいですか？

その際、〇〇さんから『ヒトモノチイキって知ってる？取材来るかもしれないけどいい人たちだよ』って先にひと言入れていただけると、すごく喜ばれると思います。」

---

### 【ステップ5】紹介者へのリターンを伝える

「ご紹介いただいた方が実際に掲載されたら、〇〇さんのことも『この方をご紹介いただきました』って投稿のキャプションに入れさせていただきます。〇〇さんのアカウントもタグ付けしますね！」

---

### 【ステップ6】思い浮かばなかった場合

「もちろん大丈夫です！あとで思い浮かんだ方がいたら、LINEでお名前だけ送ってもらえたら嬉しいです。」

## コストゼロの紹介インセンティブ案

| # | 施策 | 内容 | コスト |
|---|------|------|--------|
| 1 | **紹介者クレジット掲載** | 紹介された方の投稿キャプションに「〇〇さんよりご紹介」と記載・タグ付け | ゼロ |
| 2 | **ストーリーズ紹介者スポット** | 月1回「コミュニティビルダー」としてストーリーズで紹介 | 月30分のみ |
| 3 | **アーカイブ掲載延長** | 紹介者の投稿をハイライトに恒久保存 | ゼロ |
| 4 | **先行スクープ通知** | 次の掲載予定をLINEで事前にお知らせ | BOの工数のみ |
| 5 | **ヒトモノ大使認定** | 2名以上紹介で「ヒトモノ大使」としてプロフィール掲載 | ゼロ |

## トラッキング スプレッドシート設計

### シート1：紹介記録マスター（インタビュアーが入力）

| 列 | 項目 | 入力タイミング |
|----|------|--------------|
| A | 記録日 | インタビュー当日中 |
| B | インタビュアー名 | 同上 |
| C | インタビュー対象者名 | 同上 |
| D | 対象者Instagramアカウント | 同上 |
| E | 紹介先①　氏名 | 同上 |
| F | 紹介先①　業種・説明 | 同上 |
| G | 紹介先①　連絡先 | 同上 |
| H | 紹介先②　氏名 | 同上 |
| I | 紹介先②　業種・説明 | 同上 |
| J | 紹介先②　連絡先 | 同上 |
| K | インセンティブ付与済み | 掲載後・BO |
| L | 備考 | 随時 |

### シート2：紹介→アポ転換トラッキング（BOが管理）

| 列 | 項目 |
|----|------|
| A | 紹介先名 |
| B | 紹介元（誰から） |
| C | 連絡先 |
| D | 初回連絡日 |
| E | 初回連絡結果 |
| F | アポ確定日 |
| G | インタビュー予定日 |
| H | インタビュー完了 |
| I | FP申込 |
| J | ステータス |

### インタビュアー→BOへの通知テンプレ

```
【紹介獲得報告】
インタビュー：〇〇さん（〇〇市の革職人）
紹介先①：田中さん / 陶芸家 / 090-XXXX-XXXX
紹介先②：山本さん / パン屋 / 連絡先なし（先方から先に話してもらう）
スプシ記入済み：✅
```

## 月+20件の達成可能性

### 逆算シミュレーション

```
目標：月+20件アポ
÷ 紹介→アポ転換率：65%（紹介は温度感が高いため）
= 必要な有効紹介先：31名

÷ 連絡先取得率：70%
= 名前だけ含めた紹介先総数：44名

÷ インタビュー1件あたりの紹介数：0.6名（保守的）
= 必要インタビュー数：73件

現在のインタビュー数：90件 → 達成可能
```

### フェーズ別目標

| フェーズ | 期間 | 目標 |
|---------|------|------|
| 立ち上げ | 1ヶ月目 | スクリプト統一・スプシ運用開始・紹介発生率を計測 |
| 安定化 | 2〜3ヶ月目 | 紹介発生率30%・月+10件アポ |
| 目標達成 | 4ヶ月目〜 | 紹介発生率40〜50%・月+20件アポ |

### 成否を決める3点

1. インタビュアーが「2名指定」を毎回言い切れるか
2. 紹介情報がその日中にBOまで届くか
3. 初回コールが3日以内に動くか
