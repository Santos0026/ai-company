# Claude自動返信 品質チェックリスト

> 作成者: Mio（BO品質管理担当）
> 最終更新: 2026-04-21
> 用途: Claude AIが生成したLINEメッセージをMioがレビューする際の判断基準

---

## レビュー判定の凡例

| マーク | 意味 | 対応 |
|--------|------|------|
| ✅ 承認 | 全項目クリア。そのまま送信可 | 送信OK |
| 🟡 改善提案 | 軽微な改善推奨。任意修正 | 修正推奨・そのまま送信も可 |
| 🔴 修正必要 | ブランド基準を満たさない。修正後送信 | 修正後に再レビュー |
| ⛔ 送信禁止 | NGワード・重大リスク含む。即エスカレーション | 送信不可・Jobs報告 |

---

## チェックリスト本体

### 【A】絶対NG確認（1項目でも該当 → ⛔送信禁止）

- [ ] **A-1** クレーム・苦情・法的リスクのある内容が含まれていないか
- [ ] **A-2** 個人情報を不用意に記載・開示していないか
- [ ] **A-3** 事実と異なる情報（料金・掲載媒体・日程等）が含まれていないか
- [ ] **A-4** 「絶対に」「必ず」「保証します」など断言表現が含まれていないか
- [ ] **A-5** 顧客を急かす・プレッシャーを与える表現（「今すぐ」「早急に」）が含まれていないか

---

### 【B】NG表現・NGワード確認（1項目でも該当 → 🔴修正必要）

- [ ] **B-1** NG絵文字（🎉🎊💪👍❤️💕💖😂🤣🔥💯‼️❗🙏）が使われていないか
- [ ] **B-2** タメ口・くだけすぎた表現（「〜だよ！」「〜ね！」）が含まれていないか
- [ ] **B-3** 硬すぎるビジネス文書表現（「拝啓」「貴殿」「謹白」「誠に恐れ入りますが」）が含まれていないか
- [ ] **B-4** 命令形で終わっていないか（「〜してください。」単体で終止）
- [ ] **B-5** 「取り急ぎ」「とりあえず」などが含まれていないか
- [ ] **B-6** 「無料です！」と断言するだけの料金説明になっていないか（仕組みの説明がセットか）
- [ ] **B-7** 「大丈夫ですか？」など不安を煽る表現が含まれていないか
- [ ] **B-8** 感嘆符（！）が3つ以上連続していないか（「！！！」）

---

### 【C】絵文字ルール確認（違反 → 🔴修正必要）

- [ ] **C-1** 使用絵文字がブランド承認リスト内か（😊✨🍀🤲🌱）
- [ ] **C-2** 1メッセージの絵文字数が4個以下か
- [ ] **C-3** 絵文字は文末・段落末に配置されているか（文中に入っていないか）
- [ ] **C-4** 1段落に絵文字は1個以内か

---

### 【D】丁寧語・敬語確認（違反 → 🔴修正必要）

- [ ] **D-1** 冒頭挨拶が適切か（「お世話になっております！ヒトモノチイキ事務局でございます😊」または状況に合った挨拶）
- [ ] **D-2** 顧客の呼称が「〇〇様」になっているか（名前＋様）
- [ ] **D-3** 依頼表現が「〜くださいますよう」「〜いただけますと幸いです」など柔らかい形か
- [ ] **D-4** 締めの言葉が適切か（「どうぞよろしくお願いいたします😊」等）
- [ ] **D-5** 自社の呼称が「ヒトモノチイキ」または「ヒトモノチイキ事務局」になっているか

---

### 【E】文章構成・読みやすさ確認（違反 → 🟡改善提案）

- [ ] **E-1** 1メッセージが2〜4文程度に収まっているか
- [ ] **E-2** 1文が60文字以内程度か（長文は段落分割を推奨）
- [ ] **E-3** 句読点が適切に使われているか（句読点なしの長文になっていないか）
- [ ] **E-4** 次アクション（何をすればいいか）が明確か
- [ ] **E-5** 箇条書きを使っていないか（LINE文化に不適）

---

### 【F】ブランドトンマナ確認（違反 → 🟡改善提案）

- [ ] **F-1** 文体が「丁寧語ベース＋温かみ」になっているか（堅すぎず・くだけすぎず）
- [ ] **F-2** ヒトモノチイキのコンセプト（人と物と地域を繋ぐ）を適切に表現しているか（必要な場合）
- [ ] **F-3** 読んだ相手が「大切にされている」と感じる文体か
- [ ] **F-4** カタカナ語の多用がないか（スケジュール・コンテンツ・フォローアップ等）
- [ ] **F-5** QA_TEMPLATEに該当シーンのテンプレートがある場合、それを参照しているか

## よくあるNGパターン集（学習データ）

### NG例1：絵文字多用
```
❌ NG：
「お世話になっております😊✨ヒトモノチイキ事務局でございます🍀！
〇〇様の掲載記事は現在作成中でございます😊🌱。
完成しましたら改めてLINEよりご連絡いたします✨🤲。」
→ C-2違反（絵文字6個）、C-3違反（文中に絵文字）

✅ 修正案：
「お世話になっております！ヒトモノチイキ事務局でございます😊
〇〇様の掲載記事は現在作成中でございます。
完成しましたら改めてLINEよりご連絡いたします😊」
```

### NG例2：命令形での締め
```
❌ NG：
「日程が決まりましたらLINEでご連絡ください。」

✅ 修正案：
「日程が決まりましたらLINEにてご教示くださいますよう
よろしくお願いいたします😊」
```

### NG例3：硬すぎる表現
```
❌ NG：
「誠に恐れ入りますが、ご多用のところ恐縮ではございますが、
ご日程のご都合をお伺いできますでしょうか。」

✅ 修正案：
「お手隙の際に、ご都合のよい日時をご教示いただけますと幸いです😊」
```

### NG例4：「無料」の断言だけ
```
❌ NG：
「ご安心ください！インタビューは完全無料です！」

✅ 修正案：
「ヒトモノチイキは協賛企業様からの支援により運営しておりますため、
インタビュー取材および掲載に関して〇〇様にご費用が発生することは
基本的にはございませんのでご安心ください😊」
```

### NG例5：箇条書きの使用
```
❌ NG：
「以下のご対応をお願いいたします。
・日程を決める
・LINEでご連絡いただく
・当日オンラインでお話しする」

✅ 修正案：
「お手隙の際にご都合のよい日時をいくつかご教示いただき、
LINEにてご連絡くださいますようよろしくお願いいたします😊
当日はオンラインにてお話しさせていただきます！」
```
