# Claude AI自動返信 品質チェックリスト

**作成**: 2026-04-21 by Mio
**用途**: n8nのClaude AIが生成したLINEメッセージを送信前にチェックする際に参照

---

## チェックの流れ

```
Claude AIがメッセージを生成
        ↓
Mioの品質チェック（以下6軸）
        ↓
⛔ 送信禁止 or 🔴 修正必要 → 修正後に送信
✅ OK → そのまま送信
```

---

## 6軸チェックリスト

### A軸：絶対NG確認（⛔ 送信禁止）

- [ ] 保険・FPの詳細内容をLINEに書いていないか
- [ ] お客様の個人情報（電話番号・住所等）がメッセージに含まれていないか
- [ ] 「絶対に」「必ず」「100%」などの断言表現がないか
- [ ] 金銭的なトラブルになりうる表現がないか

→ 1つでも該当したら **⛔ 送信禁止・BOに転送**

---

### B軸：NGワード確認（🔴 修正必要）

- [ ] 「今すぐ」「急いで」「至急」がないか（プレッシャー表現）
- [ ] 「〜してください。」の単独命令形がないか
- [ ] 「取り急ぎ」がないか
- [ ] 「ご了承ください」がないか
- [ ] 「無料なので」「タダで」がないか

→ 該当した場合は **🔴 修正して送信**

**修正例：**
| NG | ✅修正後 |
|----|---------|
| 「返信してください。」 | 「返信いただけますと幸いです😊」 |
| 「今すぐご確認を」 | 「お手隙の際にご確認いただけますか？」 |
| 「取り急ぎご連絡まで」 | 「まずはご連絡いたしました😊」 |

---

### C軸：絵文字チェック（🔴 修正必要）

- [ ] 使用している絵文字がBRAND_GUIDELINEの5種（😊✨🍀🤲🌱）の範囲内か
- [ ] 1メッセージで3個以下か
- [ ] 文末のみに配置されているか（文中にないか）
- [ ] 同じ絵文字が連続していないか（😊😊はNG）
- [ ] NG絵文字（🎉❤️👍😂🙏💰）が含まれていないか

---

### D軸：敬語・呼称チェック（🔴 修正必要）

- [ ] 冒頭に「お世話になっております！ヒトモノチイキ事務局でございます😊」があるか（後追いLINEの場合）
- [ ] お客様を「〇〇様」と呼んでいるか（呼び捨てNG）
- [ ] タメ口・過度にカジュアルな表現がないか
- [ ] 「〜でございます」の適切な使用か

---

### E軸：文章構成チェック（🟡 改善提案）

- [ ] 1メッセージが5文以内か（長すぎないか）
- [ ] 1文が80文字以内か
- [ ] 適切な改行が入っているか（詰め込みすぎていないか）
- [ ] 箇条書きを過度に使っていないか（LINEでは読みにくい）

---

### F軸：トンマナ総合評価（🟡 改善提案）

- [ ] 「ヒトモノチイキらしい温かみ」があるか
- [ ] プレッシャーをかけすぎていないか
- [ ] 読んで安心できる内容か
- [ ] 「押しつけがましくない」表現になっているか

---

## よくあるNGパターンと修正例

### パターン1：冷たすぎる

❌NG:
```
山田様、日程の確認をしてください。返信をお待ちしております。
```

✅修正後:
```
お世話になっております！ヒトモノチイキ事務局でございます😊
山田様、実施アンケートの日程についてご確認のご連絡をさせていただきました。
お手隙の際にご都合の良い日時をLINEにてご教示いただけますと幸いです😊
よろしくお願いいたします！
```

---

### パターン2：絵文字が多すぎる

❌NG:
```
お世話になっております！😊✨🌱 山田様🍀
記事が完成しました！🎉✨🎊 ぜひご確認ください！😊🙏
```

✅修正後:
```
お世話になっております！
ヒトモノチイキ事務局でございます😊
山田様の掲載記事が完成いたしました✨
ぜひご確認ください！
```

---

### パターン3：断言表現

❌NG:
```
必ずご連絡いたします。絶対に対応しますのでご安心ください。
```

✅修正後:
```
確認の上、改めてご連絡いたします😊
どうぞよろしくお願いいたします。
```

---

### パターン4：命令形

❌NG:
```
アンケートフォームに入力して送信してください。
```

✅修正後:
```
お手数ですが、アンケートフォームにご入力いただき
送信いただけますと幸いです🤲
```

---

### パターン5：硬すぎる

❌NG:
```
拝啓、時下ますますご清祥のことと存じます。
さて、掲載記事についてご連絡申し上げます。
```

✅修正後:
```
お世話になっております！
ヒトモノチイキ事務局でございます😊
掲載記事についてご連絡いたします。
```
