# ヒトモノチイキ ブランドガイドライン（LINEメッセージ版）

> 作成者: Mio（BO品質管理担当）
> 最終更新: 2026-04-21
> 参照元: 実際の送信スクリプト群（初回電話・インタビュー冒頭・BO後追い・LINE自動送信・完了報告・DMスクリプト）

---

## 1. ブランドの基本思想

ヒトモノチイキは「人と物と地域を繋ぐ」コンセプトブランド。
LINEメッセージは**その接点の最前線**であり、送られた方が「丁寧に扱われている」「信頼できる」と感じる文体を常に維持する。

---

## 2. トンマナの特徴（口調・文体）

### 2-1. 丁寧度レベル

**レベル：「丁寧語ベース＋温かみ」**

- 敬語（丁寧語）を基本とする。ただし堅苦しい硬体語は避ける
- ビジネス文書調（例：「拝啓〜敬具」「誠に恐れ入りますが」）は使わない
- 話しかけるような温かみのある文体を維持する

| NG例（硬すぎ） | OK例 |
|----------------|------|
| 拝啓、時下ますますご清祥のこととお慶び申し上げます | お世話になっております！ヒトモノチイキ事務局でございます😊 |
| ご査収のほどよろしくお願い申し上げます | どうぞよろしくお願いいたします✨ |
| ご多忙のところ恐縮ではございますが | お手隙の際に〜 |

### 2-2. 話し言葉と書き言葉の割合

**「書き言葉ベース＋会話的テンポ」**

- 文章の骨格は丁寧な書き言葉
- 接続詞や語尾に会話的なリズムを入れる（「で、ですね」「あっなるほど」など電話スクリプトでは顕著）
- LINEメッセージでは書き言葉中心だが、1文を短めに区切り読みやすくする

### 2-3. 文章の長さの傾向

- **1メッセージ = 2〜4文が基本**
- 1文は30〜60文字程度が目安
- 長くなる場合は段落を分ける（1メッセージで2〜3段落まで）
- 箇条書きは基本使わない（LINE文化に合わない）

### 2-4. 特徴的な言い回し（使うべき表現）

| 表現 | 使用場面 |
|------|----------|
| お世話になっております！ | メッセージ冒頭の挨拶 |
| ヒトモノチイキ事務局でございます | 自己紹介・送信者名 |
| お手隙の際に | 依頼するとき（急かさない配慮） |
| ご教示くださいますよう | 情報を求めるとき |
| 改めてLINEよりご連絡いたします | 次アクションの案内 |
| 素敵な御縁がありますように | 締めの言葉（特に初回） |
| どうぞよろしくお願いいたします | 締めの言葉（汎用） |
| 〜でございます | 断定するとき（「〜です」より丁寧） |
| 〜くださいますよう | お願いするとき（「〜してください」より柔らかい） |

### 2-5. 呼称ルール

- 顧客には必ず「〇〇様」と名前＋様をつける
- 自社は「ヒトモノチイキ」または「ヒトモノチイキ事務局」と表記（略さない）
- 「弊社」は電話スクリプトでは使用可。LINEメッセージでは「ヒトモノチイキ」に統一推奨

## 3. 絵文字ルール

### 3-1. 使用する絵文字と使用シーン

| 絵文字 | 意味・役割 | 主な使用場面 |
|--------|-----------|-------------|
| 😊 | 温かみ・親しみ・笑顔 | メッセージ末尾全般、自己紹介後 |
| ✨ | 輝き・特別感・期待感 | 締めの言葉（素敵な御縁〜等）、ポジティブな内容 |
| 🍀 | 幸運・縁起・繋がり | DMの冒頭挨拶、締めの言葉 |
| 🤲 | 依頼・お願い・受け取る | アンケート依頼、お願い事項 |
| 🌱 | 成長・始まり・自然 | アンケートフォーム等、新しい取り組みの案内 |

### 3-2. 1メッセージあたりの絵文字使用数

| メッセージタイプ | 目安個数 |
|----------------|---------|
| 短いメッセージ（1〜2文） | 1個 |
| 通常メッセージ（2〜4文） | 1〜2個 |
| 長めメッセージ（4文以上） | 2〜3個（上限） |
| DMの冒頭メッセージ | 2個（冒頭1＋末尾1） |

**絵文字過多のNG基準：1メッセージに4個以上は多すぎる**

### 3-3. 絵文字の位置ルール

- **文末が基本**（文中に絵文字を入れない）
- 1段落に絵文字は最大1個
- 文章の最後の文末、または段落の末尾につける

```
✅ OK例：
「〇〇様の掲載記事は現在作成中でございます。
完成しましたら改めてLINEよりご連絡いたします😊」

❌ NG例：
「〇〇様😊の掲載記事は現在作成中でございます✨。
完成しましたら改めてLINEよりご連絡いたします🍀✨😊」
```

### 3-4. 絵文字の組み合わせパターン（推奨）

| 場面 | 推奨パターン |
|------|-------------|
| 通常の締め | 〜よろしくお願いいたします😊 |
| 初回・嬉しい知らせ | 〜よろしくお願いいたします✨🍀 |
| 依頼・お願い | 〜よろしくお願いいたします🤲 |
| DM冒頭 | こんにちは😊〜（冒頭）〜よろしくお願いいたします✨🍀（末尾） |

## 4. NGリスト

### 4-1. NG絵文字

| 絵文字 | NGの理由 |
|--------|---------|
| 🎉🎊 | お祝い感が強く、ビジネス文脈では浮く |
| 💪👍 | カジュアルすぎる・男性的な印象 |
| ❤️💕💖 | 恋愛的なニュアンスが出る |
| 😂🤣 | 笑いを誇張する表現。信頼感を損なう |
| 🔥💯 | 煽り・押しつけ感が出る |
| ‼️❗ | 強調しすぎ。圧力的に感じる |
| 🙏 | 宗教的ニュアンスが含まれる場合がある（🤲で代替） |

### 4-2. NG表現・NGワード

| NG表現 | 理由 | 代替表現 |
|--------|------|----------|
| 絶対に・必ず | 保証・断言の印象。トラブル時のリスク | 基本的には・通常は |
| 無料です！ | 過度に強調しすぎ・押し売り感 | 料金はかかりませんのでご安心ください |
| 今すぐ・急いで | 急かす・プレッシャーを与える | お手隙の際に |
| ご連絡ください（命令形） | やや強い命令口調 | ご連絡いただけますと幸いです |
| 〜してください。（命令形で終止） | 指示的すぎる | 〜くださいますようよろしくお願いいたします |
| 大丈夫ですか？ | 不安を煽る印象 | 何かご不明な点などございましたら |
| 取り急ぎ | 雑・粗略な印象 | 改めて・後ほど詳しく |
| 〜の方（「LINEの方に」など） | 文法的曖昧さ（許容するが多用注意） | 〜に・〜へ（ただし「LINEの方に」は既成表現として容認） |

### 4-3. NG口調・NG文体

| NG | 理由 |
|----|------|
| タメ口・くだけすぎた表現（「〜だよ！」「〜ね！」） | ブランドの品位を損なう |
| 硬すぎるビジネス文書体（「貴殿」「拝啓」「謹白」） | 距離感が出て温かみがなくなる |
| 過度な自己卑下（「私どもの不手際で〜」等のクレーム対応語） | 信頼を損なう場合がある（ミス時は適切に使用可） |
| 「！！！」多用 | 興奮・押しつけ感が出る |
| カタカナ語の多用（「スケジュール」「コンテンツ」等） | 一部は許容だが多用は文体を乱す |
| 句読点なしの長文 | 読みにくい・信頼感低下 |

## 5. シーン別テンプレート文体チェック

### 初回コンタクト（DM・アウトバウンド）
- 冒頭に「こんにちは😊」
- 自己紹介（名前・ヒトモノチイキ）
- 用件は簡潔に（無料であることを自然に）
- 行動喚起は「興味あり」とだけ返信→心理的ハードルを下げる
- 締めは「素敵な御縁がありますように✨🍀」

### 進捗報告・ステータス連絡
- 冒頭に「お世話になっております！ヒトモノチイキ事務局でございます😊」
- 現状＋次アクションをセットで伝える
- 締めは「どうぞよろしくお願いいたします😊」

### 依頼・お願い
- 「お手隙の際に〜」で急かさない
- 「〜くださいますようよろしくお願いいたします😊」で丁寧に締める

### アンケート・フォーム誘導
- 「〜"完了報告アンケートフォーム"になります」
- 「〜ご入力いただき送信ください🤲🌱」
- 「引き続きどうぞ宜しくお願いいたします🤲🍀」
