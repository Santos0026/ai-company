# ヒトモノチイキ 公式LINE ブランドガイドライン

**作成**: 2026-04-21 by Mio（BO品質管理担当）
**抽出元**: 実際のスクリプト・提携文・DMテンプレート群

---

## 1. 口調・文体の基本原則

### 丁寧度レベル：「丁寧語ベース＋温かみ」

| NG（硬すぎる） | ✅OK | NG（やわらかすぎる） |
|--------------|------|------------------|
| 「拝啓」「謹白」 | 「お世話になっております！」 | タメ口 |
| 「〜致しかねます」 | 「〜させていただきます😊」 | 「〜だよ」「〜ね！」 |
| 「ご査収ください」 | 「ご確認いただけますか？」 | 過度なため語 |

### 特徴的な言い回し（使ってよい表現）

- 「お世話になっております！」（冒頭定形）
- 「〜でございます😊」
- 「〜よろしくお願いいたします😊」
- 「お手隙の際に〜」
- 「ご教示くださいますよう」
- 「引き続きどうぞよろしくお願いいたします🤲🍀」
- 「素敵な御縁がありますように✨🍀」（DM締め専用）

### 文章の長さ

- 1メッセージ：2〜5文を目安
- 1文：30〜60文字程度
- 長くなる場合は改行で区切る（詰め込まない）

## 2. 絵文字・記号のルール

### 使用する絵文字5種（用途別）

| 絵文字 | 使うシーン | 使用頻度 |
|--------|-----------|---------|
| 😊 | 全般・文末・親しみを表す | 最多（1メッセージ1〜2個） |
| ✨ | ポジティブな内容・締めの言葉 | 1〜2個/メッセージ |
| 🍀 | 縁起・初回DM・締め | DM専用・1個まで |
| 🤲 | 依頼・お願い・感謝 | 依頼文の末尾に1個 |
| 🌱 | 新しい取り組み・成長・自然 | 限定的使用 |

### 絵文字配置ルール

- **文末のみ**に置く（文中に絵文字を挟まない）
- **1段落に1個**まで
- **1メッセージ最大3個**まで
- 同じ絵文字を連続して使わない（😊😊 はNG）

### 使ってはいけない絵文字

❌ 🎉🎊（派手すぎる・ビジネス文脈でNG）
❌ 💰💸（金銭的なイメージ）
❌ ❤️💕（恋愛・親密すぎる）
❌ 😂🤣（笑いすぎ・失礼な印象）
❌ 👍（上から目線な印象）
❌ 🙏（宗教的・過度な謝罪感）

## 3. NGワード・NGパターン

### 絶対NG（送信禁止）

- 「絶対に」「必ず」「100%」（断言表現）
- 「今すぐ」「急いで」（プレッシャーを与える）
- 「無料なので」「タダで」（価値を下げる表現）
- 保険・FP関連の話をLINEで詳述する（電話・インタビューでのみ行う）
- お客様の個人情報をLINEに記載する

### 要注意（文脈による）

- 「取り急ぎ」（温かみがない）→「お急ぎのところ恐縮ですが」
- 「〜してください。」単独命令形 → 「〜いただけますと幸いです😊」
- 「ご了承ください」（一方的）→「ご理解いただけますと幸いです」
- 「〜の件」だけで始める → 「先日の〇〇についてご確認のご連絡です！」

## 4. シーン別の口調指定

### 初回DM

明るく・短く・ハードルを下げる
```
こんにちは😊
突然のDM失礼いたします🍀
[本文：2〜3文]
もし少しでもご興味あれば、『興味あり』とだけ返信していただければ大丈夫です！
素敵な御縁がありますようにどうぞよろしくお願いいたします✨🍀
```

### 後追い・進捗確認

安心感を与える・プレッシャーをかけない
```
お世話になっております！
ヒトモノチイキ事務局でございます😊
[確認内容：1〜2文]
お手隙の際にご確認いただけますと幸いです😊
引き続きよろしくお願いいたします！
```

### アンケート・お願いごと

感謝をセットにする・理由を添える
```
[感謝の一文]
お手数をおかけいたしますが、〜いただけますと幸いです🤲
[締めの一文]
引き続きどうぞよろしくお願いいたします🤲🍀
```

### 謝罪・お詫び

迅速・誠実・解決策をセットにする
```
大変失礼いたしました🙇
[謝罪内容：1文]
[対処内容：1文]
引き続きよろしくお願いいたします😊
```

## 5. 「ヒトモノチイキらしさ」の本質

スクリプト全体を通じて一貫しているもの：

1. **押しつけがましくない** — 「大丈夫です！」「幸いです」「いつでも」という表現が多く、相手にプレッシャーをかけない
2. **温かみと礼儀が共存** — 敬語を使いながら感情的な温かさ（絵文字・語尾）を忘れない
3. **短く・分かりやすく** — 難しい言葉を使わず、読みやすい
4. **誠実さ** — 「確認します」「連絡します」と約束したことは明確に伝える
