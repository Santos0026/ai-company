## Q1：「掲載はいつになりますか？」「記事の進捗は？」

### 状況A：まだ作成中

```

完成しましたらこちらのLINEにてご連絡いたしますので、
もうしばらくお待ちいただけますと幸いです🌱
引き続きよろしくお願いいたします！
```

### 状況B：最終確認待ち

```
お問い合わせいただきありがとうございます😊
〇〇様の掲載記事はただいま最終確認中でございます！
間もなくLINEにてご確認いただける記事をお送りいたします。
もう少々お待ちください😊
```

### 状況C：掲載済み

```
お問い合わせいただきありがとうございます😊
〇〇様の記事はこちらでご覧いただけます✨
[Instagram URL]
ぜひご確認いただけますと嬉しいです😊
```

## Q10：エスカレーション（自動回答できないケース）

以下の場合は自動回答せず、BOスタッフに転送する：

- 法的なトラブル・クレーム
- 保険面談の詳細についての質問（担当FPへ）
- 「詐欺ではないか」という疑念
- 個人情報の変更・削除依頼
- 金銭に関するトラブル

```
【BOスタッフへの転送通知】
⚠️ 自動回答不可のメッセージです
ユーザー: [LINE表示名]
内容: [メッセージ内容]
→ 担当スタッフが対応してください
```
