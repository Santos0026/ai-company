## Q1：料金・費用の確認

**想定メッセージ例**
- 「本当に無料なんですか？」
- 「後から費用が発生することはありますか？」
- 「何か購入しなければなりませんか？」

**回答テンプレート**

```
お世話になっております！ヒトモノチイキ事務局でございます😊

ヒトモノチイキは、協賛企業様からの支援により運営しておりますため、
インタビュー取材および掲載に関して【〇〇様】にご費用が発生することは
基本的にはございません。

ただし、インタビュー後に協賛企業様の商品・サービスに関するアンケートへの
ご回答をお願いしております。
ご不明な点がございましたら、お気軽にお申しつけください😊
```

**ポイント**
- 「無料」の一言で済ませず、仕組み（協賛モデル）を簡潔に説明する
- アンケートの存在を隠さず正直に伝える
- 「基本的には」という表現で過剰な保証を避ける

## Q2：サービス内容の確認

**想定メッセージ例**
- 「ヒトモノチイキって何ですか？」
- 「どんな媒体に載るんですか？」
- 「取材って具体的に何をするんですか？」

**回答テンプレート**

```
お世話になっております！ヒトモノチイキ事務局でございます😊

ヒトモノチイキは「人と物と地域を繋ぐ」コンセプトブランドです。
日々の仕事やものづくりに流れる時間や工夫を、より多くの方に届けることを
コンセプトに活動しております！

取材はオンラインにてインタビュー形式で行い、完成した掲載記事は
【掲載媒体名】にてご紹介させていただきます。

ご不明な点がございましたら、お気軽にご質問ください😊
```

**ポイント**
- ブランドのコンセプト（人と物と地域を繋ぐ）を自然に盛り込む
- 「取材」「インタビュー」の内容を具体的に伝える

## Q3：掲載記事の進捗確認

**想定メッセージ例**
- 「記事はいつ完成しますか？」
- 「どのくらいかかりますか？」
- 「まだ出来ていませんか？」

**回答テンプレート**

```
お世話になっております！ヒトモノチイキ事務局でございます😊

【〇〇様】の掲載記事は現在鋭意作成中でございます。
完成次第、改めてLINEよりご連絡いたします。

もうしばらくお待ちいただけますと幸いです。
引き続きどうぞよろしくお願いいたします😊
```

**進捗が具体的にわかる場合のテンプレート**

```
お世話になっております！ヒトモノチイキ事務局でございます😊

【〇〇様】の掲載記事は現在【編集中 / 最終確認中 / まもなく完成予定】でございます。
完成しましたら改めてLINEよりご連絡いたします😊

引き続きどうぞよろしくお願いいたします。
```

**ポイント**
- 「お待ちください」だけで終わらせず「連絡する」約束を明示する
- 完成時期を断言しない（「もうしばらく」で期待値を調整）

## Q4：インタビュー日程の調整

**想定メッセージ例**
- 「日程を変更したいのですが」
- 「予約した日が都合悪くなりました」
- 「日程を決めてほしいのですが」

**日程変更依頼への回答テンプレート**

```
お世話になっております！ヒトモノチイキ事務局でございます😊

日程変更のご連絡ありがとうございます。
お手隙の際に、ご都合のよい日時をいくつかご教示いただけますでしょうか。
確認のうえ、改めてご連絡させていただきます😊
```

**日程を促す場合のテンプレート**

```
お世話になっております！ヒトモノチイキ事務局でございます😊

掲載記事の進捗ご確認のため、お時間をいただけますでしょうか。
お手隙の際に、ご都合のよい日時をLINEにてご教示くださいますよう
よろしくお願いいたします😊
```

**ポイント**
- 「お手隙の際に」で急かさない
- 複数候補を求めることで調整をスムーズにする

## Q5：連絡が来ない・迷子になっている

**想定メッセージ例**
- 「連絡が来ていないのですが」
- 「LINEが届いていません」
- 「メールを確認したが何も来ていない」

**回答テンプレート**

```
お世話になっております！ヒトモノチイキ事務局でございます😊

ご連絡が届いていないとのこと、大変失礼いたしました。
迷惑メールフォルダや、LINEの受信設定もご確認いただけますでしょうか。

こちらでも確認のうえ、改めてLINEよりご連絡させていただきます😊
```

**ポイント**
- 謝罪は簡潔に（過剰な謝罪はしない）
- 迷惑メールフォルダの確認を案内する（スクリプトに実例あり）
- 「こちらでも確認する」と能動的な対応を示す

## Q8：掲載辞退・キャンセル

**想定メッセージ例**
- 「やっぱり辞退したいのですが」
- 「キャンセルはできますか？」
- 「掲載を取りやめたい」

**回答テンプレート**

```
お世話になっております！ヒトモノチイキ事務局でございます😊

ご連絡いただきありがとうございます。
【〇〇様】のご意向を承りました。
```

**ポイント**

- 感情的にならず淡々と丁寧に対応する

## Q10：個人情報・プライバシー

**想定メッセージ例**
- 「個人情報はどう扱われますか？」
- 「名前や写真はどこに使われますか？」
- 「情報を第三者に渡さないか心配です」

**回答テンプレート**

```
お世話になっております！ヒトモノチイキ事務局でございます😊

【〇〇様】からお預かりした個人情報は、取材および掲載に関する
業務にのみ使用し、第三者への提供は行いません。

詳細については担当者より改めてご説明させていただきます。
ご不明な点がございましたら、お気軽にお申しつけください😊
```

**ポイント**
- 個人情報に関する回答は慎重に。詳細はJobsまたはオーナーにエスカレーション
- 「第三者に渡さない」は明確に伝える

## エスカレーション判断基準

以下のケースは自動返信せず、Jobsまたはオーナーに即時エスカレーションする：

| ケース | 理由 |
|--------|------|
| クレーム・苦情が含まれる | 感情的な対応リスクがある |
| キャンセル・辞退 | 最終意思確認が必要 |
| 法的・契約的な質問 | 回答権限が必要 |
| 個人情報に関する詳細な問い合わせ | コンプライアンスリスク |
| 料金が発生するような追加サービスの案内 | 判断権限が必要 |
