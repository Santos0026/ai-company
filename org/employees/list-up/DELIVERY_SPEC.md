## 優先順位のルール

```
Aランク（最優先）：公式LINEリンク（lin.ee）あり
    → LINEでDM送信

Bランク：Instagramリンクあり（LINEなし）
    → Instagram DMで送信

Cランク：電話番号のみ（SNSリンクなし）
    → 納品対象外（代理店が使えない）
```

**CランクはLunaの納品リストから除外する。**

## エキテンの課題と対応

エキテンの一覧ページには公式LINEリンクが表示されない。
LINEリンクは**各店舗の詳細ページ**にのみ掲載される。

### 対応方法（優先度順）

**方法①：Octoparseの2段階クロール**
- 一覧ページ → 詳細ページ → lin.ee URLを取得
- 精度が高いが設定工数が必要

**方法②：エキテン詳細ページを手動確認（初回のみ）**
- 一覧で取得した店舗をサンプル確認
- lin.ee掲載率・業種別傾向を把握してから自動化

**方法③：Instagramリンク優先で先に納品**
- エキテンに掲載されているInstagramリンクを取得
- LINEリンクは別途Googleマップ・店舗HPから補完
