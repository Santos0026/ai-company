# 新AI社員オンボーディングガイド

このガイドはJobsが新しいAI社員を採用・設定する際の手順書です。

---

## ステップ1: ディレクトリ作成

```bash
mkdir -p org/employees/<role-name>/logs
```

## ステップ2: キャラクター定義ファイルの作成

`org/employees/template/CHARACTER.md` をコピーして編集します。

以下の項目を必ず埋めてください：
- [ ] 名前
- [ ] 役職
- [ ] 管轄領域
- [ ] コアアイデンティティ（3段落）
- [ ] 強み（最低3つ）
- [ ] 行動指針（最低4つ）
- [ ] やらないこと（最低3つ）
- [ ] 初期ミッション

## ステップ3: MEMORY.md の初期化

```markdown
# [名前] — 記憶・蓄積コンテキスト
## 会社の基本情報（CLAUDE.mdから転記）
## 確定した意思決定
## 進行中のアジェンダ
```

## ステップ4: CURRENT_FOCUS.md の初期化

最初の1〜3タスクを設定します。

## ステップ5: CLAUDE.md の組織構成を更新

`/workspace/CLAUDE.md` の組織図に新社員を追記します。

## ステップ6: Jobsからのウェルカムメッセージ

`logs/YYYY-MM-DD_onboarding.md` にオンボーディング記録を作成します。

---

## チェックリスト

- [ ] `CHARACTER.md` 作成・完成
- [ ] `MEMORY.md` 作成
- [ ] `CURRENT_FOCUS.md` 作成
- [ ] `logs/` ディレクトリ作成
- [ ] `CLAUDE.md` 組織構成更新
- [ ] オンボーディングログ作成

---
