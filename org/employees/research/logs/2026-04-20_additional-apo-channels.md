### 【手法1】掲載済み事業者のInstagramコメント欄から同業者を発掘

**優先度：S（最高）**

#### 概要
ヒトモノチイキのInstagramに掲載した事業者の投稿に「すごいですね！」「うちもやりたい」等のコメントをつけている他アカウントを特定し、直接アプローチする。
掲載事業者の仲間・知人・同業者がコメントしている確率が高く、**すでに掲載事業者を通じて「信頼の橋渡し」が済んでいる**状態に等しい。

#### ロジック
- 掲載投稿のリーチは事業者フォロワー中心
- コメントする人 = 掲載に興味がある / 事業者仲間である確率が高い
- 「○○さんの投稿でヒトモノチイキさんを知りました」という文脈でDMできる → 返信率が通常DMより高い（仮説：返信率8〜12%想定）

#### 数値試算

| 指標 | 数値 |
|------|------|
| 掲載投稿数（累計） | 推定103件以上 |
| 1投稿あたりコメント者数 | 平均3〜5件 |
| 事業者コメント者割合 | 約40% |
| 月間発掘可能数 | 新規投稿10件×5コメント×40% ≈ 20件/月 |
| DM→アポ転換率（仮説） | 10%（通常の2倍以上） |
| **月間期待アポ数** | **+20件** |

#### コスト・工数

| 項目 | 内容 |
|------|------|
| 金銭的コスト | 0円 |
| 工数 | 30分/日（Hanaがコメント欄を巡回・リスト化・KaiへDM引き渡し） |

#### 担当
- **Hana**（コメント欄の発掘・Instagramアカウント特定）
- **Kai**（DM送信管理）

#### 初日アクション
1. Hanaが過去30日分の掲載投稿（直近10件）のコメント欄を全件チェック
2. 個人事業主と判断できるアカウントをリスト化（業種・@URL・フォロワー数記載）
3. KaiへDMリストとして渡す（通常ルートで送信）

### 【手法2】掲載事業者による紹介制度（リファラルプログラム）

**優先度：S（最高）**

#### 概要
掲載完了後・インタビュー後の事業者に「仲の良い同業者・近所の事業主を1〜2名紹介してください」と依頼するフローを標準化する。
現在103件の掲載実績 = **103名の潜在アンバサダーが存在するにもかかわらず、このルートが未活用**である点が最大の機会損失。

#### ロジック
- 掲載済み事業者は「ヒトモノチイキに掲載してもらって良かった」という実体験を持つ
- その人が「友達のネイリストも紹介していい？」と言うだけでアポが1件生まれる
- 信頼コストが限りなく低い（0に近い）→ 転換率は通常DMの3〜5倍が見込める

#### 数値試算

| 指標 | 数値 |
|------|------|
| 掲載済み事業者数（累計） | 103件 |
| 紹介依頼実施率（想定） | 月20件分に声がけ |
| 紹介が出る割合 | 30%（6件） |
| 紹介→アポ転換率 | 60%（インタビュー前提のため高い） |
| **月間期待アポ数** | **+15件** |

#### コスト・工数

| 項目 | 内容 |
|------|------|
| 金銭的コスト | 0円（現在の掲載フローに一言追加するだけ） |
| 工数 | 掲載完了時のトークスクリプトに1行追加（5分/件） |

#### 担当
- **Kai**（掲載完了後の紹介依頼スクリプト管理・紹介リスト追跡）
- **Ryo**（紹介依頼トークスクリプトの設計）

#### 初日アクション
1. Ryoが「紹介依頼トークスクリプト」を3パターン作成（LINEテキスト版）
2. Kaiが掲載完了後のフローに「紹介依頼ステップ」を追加
3. 既存の掲載済み103件に対しても遡及して依頼（週20件ずつ）

### 【手法3】TikTok・YouTube Shortsで「掲載体験談」を発信しインバウンド獲得

**優先度：A**

#### 概要
掲載済み事業者の「無料掲載を受けた感想」や「インタビューの様子」を15〜60秒の縦型動画で発信する。
「こんな無料サービスがあるの知らなかった」「うちもやってほしい」という反応者が問い合わせてくる**完全インバウンド型**のチャネル。

#### ロジック
- TikTokアルゴリズムは地域×業種のニッチコンテンツが拡散されやすい
- 「無料でインタビューしてもらった」という体験談は事業者から見て価値ある情報
- 1本のバイラル動画が100〜200件の問い合わせを生む可能性がある（単発但し大きなアップサイド）
- YouTube Shortsはタイムスタンプが残るためGoogleインデックスされ、長期的に流入が続く

#### 数値試算（保守的）

| 指標 | 数値 |
|------|------|
| 月間投稿本数 | 15本（週3〜4本） |
| 平均再生数 | 1,000〜3,000回/本 |
| 問い合わせ転換率 | 0.3% |
| **月間期待アポ数** | **+15〜45件（中央値20件）** |

#### コスト・工数

| 項目 | 内容 |
|------|------|
| 金銭的コスト | 0円（スマホ1台でOK・既存インタビュー素材を再活用） |
| 工数 | 編集30分/本×週3本 = 週1.5時間（人間側の実作業） |
| 立ち上げコスト | 最初の5本の方向性設計（Ryo）：1〜2日 |

#### 担当
- **Ryo**（コンテンツ戦略・投稿カレンダー設計）
- **新規「Nao」（動画コンテンツ担当）**を設置するか、Soraの役割を拡張して担当させる
- 動画撮影・編集はSI Partners（人間）が実行

#### 注意点
- 掲載事業者の許可を取ること（顔出し・店名出しの確認）
- 「広告っぽくない」自然な体験談として仕上げることが鍵

#### 初日アクション
1. Ryoがコンテンツ企画書を作成（タイトル案10本・動画構成テンプレート）
2. Soraが過去の掲載事業者から「動画出演OK」な人を3名リストアップ
3. TikTok・YouTube Shortsのビジネスアカウント開設

### 【手法4】地域の商工会・BNI・異業種交流会への定期参加

**優先度：A**

#### 概要
地域の商工会・商工会議所・BNI（ビジネス紹介ネットワーク）・地域の異業種交流会に月2〜4回参加し、**一度に複数の個人事業主に「無料掲載企画」を説明**する。
1回の参加で10〜30名の事業者と接触でき、DMを送るよりも受け入れられやすい。

#### ロジック
- 商工会メンバー = 個人事業主・小規模事業者がほぼ100%
- 「名刺交換ついでにLINE交換」で自然に連絡先を取得できる
- 「こういうメディアで無料で取り上げてもらえる」という文脈は交流会で非常に刺さりやすい
- BNIは「他者を紹介する」文化があるため、一人説得すると連鎖的に広がる

#### 数値試算

| 指標 | 数値 |
|------|------|
| 月間参加回数 | 4回 |
| 1回あたり接触数 | 15名 |
| LINE交換率 | 50%（7〜8名） |
| LINE→アポ転換率 | 25%（現状の約6倍・対面後のため高い） |
| **月間期待アポ数** | **+10〜12件** |

#### コスト・工数

| 項目 | 内容 |
|------|------|
| 金銭的コスト | 参加費500〜3,000円/回（無料の会も多い） |
| 工数 | 移動込み3時間/回×4回 = 月12時間（人間側実作業） |

#### 担当
- **Ryo**（各地域の対象コミュニティのリサーチ・参加申込管理）
- **実行はSI Partners（人間）**が参加・名刺交換を担う
- 参加後のフォローはKaiが管理

#### 初日アクション
1. Ryoが福岡市・東京の商工会・BNI・無料交流会の開催情報をリサーチ（一覧化）
2. 今月中に参加できる交流会を2件以上ピックアップしてJobsに提案
3. 持参するチラシ・QRコードの資料をRyoが企画・Kai/Soraで内容を精査

### 【手法5】Facebook・地域Facebookグループへの掲載事例投稿

**優先度：A**

#### 概要
「○○市の個人事業主さんの会」「福岡起業家コミュニティ」等の地域Facebookグループに、ヒトモノチイキの掲載事例を定期投稿する。
グループメンバーは個人事業主が多く、「私もやりたい」というコメントやDMが自然発生する**完全インバウンド型**チャネル。

#### ロジック
- Facebookグループの地域コミュニティは現在も活発（特に地方・30〜50代の事業主）
- 「無料でインタビューしてもらった事業者の声」は広告でなく情報として受け入れられる
- グループ管理者に許可を取れば、定期的に投稿できる枠が確保できる
- 1投稿で100〜300名にリーチ可能（グループ規模次第）

#### 数値試算

| 指標 | 数値 |
|------|------|
| 月間投稿グループ数 | 10グループ（東京5・福岡5） |
| 1グループあたり月2投稿 | 計20投稿/月 |
| 1投稿あたりDM問い合わせ | 1〜2件 |
| 問い合わせ→アポ転換率 | 40%（インバウンドのため高い） |
| **月間期待アポ数** | **+10〜16件** |

#### コスト・工数

| 項目 | 内容 |
|------|------|
| 金銭的コスト | 0円 |
| 工数 | グループ開拓2時間/週（初期）・投稿管理1時間/週（継続） |

#### 担当
- **Hana**（Facebookグループの発掘・参加申請・投稿文作成）
- **Kai**（問い合わせ対応・アポ管理）

#### 初日アクション
1. Hanaが東京・福岡の地域Facebookグループを20件リサーチしてリスト化
2. 各グループに参加申請（個人アカウントで自然に参加）
3. 最初の投稿文テンプレートをRyoが作成（掲載事例紹介型・親しみやすい文体）

### 【手法6】ホットペッパービューティー・楽天ビューティー掲載店への直接アプローチ

**優先度：B**

#### 概要
ホットペッパービューティーや楽天ビューティーに掲載されている美容室・ネイル・エステに絞り、
プラットフォームのメッセージ機能またはInstagram経由でアプローチする。
これらのプラットフォーム掲載店は**SNS活用意識が高く・Instagram連携率が80%超**のため、転換率が高い。

#### ロジック
- ホットペッパービューティー掲載 = 集客に積極的・広告費を使っている
- 「今より低コストで認知を取れる方法」には即反応しやすい
- 美容系はFP面談の転換率が実績上も高い業種（MEMORY.mdの知見）
- 特に個人経営サロン（スタッフ数1〜3名）が最優先ターゲット

#### 数値試算

| 指標 | 数値 |
|------|------|
| 月間アプローチ数 | 500件（Kentoがリスト作成） |
| 返信率 | 3〜5%（15〜25件） |
| 返信→アポ転換率 | 60% |
| **月間期待アポ数** | **+10〜15件** |

#### コスト・工数

| 項目 | 内容 |
|------|------|
| 金銭的コスト | 0円 |
| 工数 | Kentoが月500件リスト作成（週125件・Luna並みの量） |

#### 担当
- **Kento**（ホットペッパービューティーからのリスト取得・Instagram URL付きで納品）
- **Kai**（DM送信管理）

#### 初日アクション
1. Kentoが「東京×美容室×個人サロン」でホットペッパービューティーからリスト100件取得
2. Instagram URLを付加してKaiへ納品

## Ryoの所感・懸念点

### 最も即効性が高いのは「紹介制度（S-2）」と「コメント欄発掘（S-1）」
- どちらも追加コスト0円・既存オペレーションの延長線上
- **今日から動ける**のに未着手だったことが最大の機会損失
- 2手法合わせて月+35件のアポが見込め、売上換算で約9万円（@16,100×52%申込率×35件）

### 中期（1〜2ヶ月後）の主力候補は「TikTok/Shorts（A-1）」と「Facebookグループ（A-3）」
- 一度仕組みができると、継続的なインバウンドが発生する
- LINEアウトバウンドのようなBANリスクがない完全コンプライアンスチャネル

### 未検証のリスク
- TikTok：業種によっては保険関連コンテンツに制限がかかる場合がある → FP面談の訴求ではなく「無料掲載メディア」の認知のみを訴求する形で設計すること
- 商工会：参加者が同業（保険・FP）の場合は競合になるため、参加前に構成を確認すること
