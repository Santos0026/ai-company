**日付**: 2026-04-17

---

## 重大発見：現行コールドDMのリスク

MetaのGraph APIでは「ユーザーが先にアクションしていない相手へのDM送信」は禁止されている。
SI Partnersの現行手法は非公式ツール依存の可能性があり、2026年現在、アカウント凍結リスクが増大している。

**→ SI Partnersに依存しない合法的な集客エンジンを並行構築する価値が高い。**

## 各チャネル詳細

### チャネル①：Instagram コメント→DM自動化（Inrø）

**ツール：** Inrø（inro.social）— Meta認証済み・AI Agent搭載
**コスト：** フリープランで月100件まで無料（有料は月€12.99〜）

フロー：
```
投稿「詳細は『知りたい』とコメントしてください」
→ コメント発生
→ Inrøが自動DM「LINE友達追加でご案内します → [lin.ee/〜]」
→ LINE友達追加
```

従来コールドDM比較：
- 開封率：20〜30% → 60〜80%（2〜4倍）
- ToSリスク：高 → ゼロ（Meta認証）

---

### チャネル②：Threadsコメント→Claude自動返信（n8n）

**ツール：** Threads API（Meta公式）+ n8n + Claude API
**コスト：** n8n VPS月$3 + Claude API月500〜1,000円

フロー：
```
Threads投稿（月・水・金）
→ コメント発生
→ Webhook → n8n
→ Claude AIが返信文生成
→「詳細はLINEで→ [URL]」と自動返信
```

**注意：** Threads APIのApp Review審査に2〜4週間かかる。今週申請開始が必要。

---

### チャネル③：LINEチャットボット（n8n × Claude × Calendly）

**コスト：** LINE公式ライトプラン月5,000円 + n8n月500円 + Claude API月500〜1,000円 = 月6,000〜7,000円

フロー：
```
LINE友達追加
→ Claude AIが3問ヒアリング
   Q1: Instagramアカウントはお持ちですか？
   Q2: お仕事の業種を教えてください
   Q3: FP（保険相談）へのご協力はOKですか？
→ 条件合致 → Calendlyリンクを自動送付
→ 日程選択 → LINEに確認メッセージ自動送信
→ アポ確定
```
