## ①Instagram DM自動化ツール評価

### 大前提：コールドDMはMeta公式APIで不可能

**調査結果（重要）：**  
Meta Graph API経由では、**自分から見知らぬユーザーへのDM（コールドDM）送信は一切できない**。これはAPIの設計上の制限であり、2026年現在も変わっていない。

| 制約 | 内容 |
|------|------|
| 24時間ルール | ユーザーが先にアクション（コメント・DM・ストーリー返信・広告クリック）した後24時間以内のみメッセージ可能 |
| 送信上限 | 200〜5,000DM/時間（アカウント規模・エンゲージメントによる） |
| コールドDM | 完全に禁止。APIでは技術的に実行不能 |

→ **SI Partnersが現在行っている200通/日の手動コールドDMは、公式ではなく非公式ツール（ブラウザ操作自動化等）に依存している可能性が高い。Metaに検出され次第アカウント凍結リスクあり。**

### 代替：「コメント→DM自動化」（合法・高効率）

| 手法 | 仕組み |
|------|--------|
| コメントキーワードDM | 投稿に「コメントDMで無料掲載の詳細をお届け！」と書く → ユーザーがコメントする → 自動でDMを送信 |
| 効果 | コールドDMの10〜40倍の開封率（60〜80% vs 20〜30%） |
| 合法性 | Meta公式Graph API使用・ポリシー準拠 |

### 推奨ツール比較

| ツール | 月額 | 特徴 | 評価 |
|--------|------|------|------|
| **Inrō** | €0〜€12.99（約0〜2,000円） | Meta認証済み・AI Agent搭載・コメント→DM→会話継続まで一気通貫 | ◎ **最推奨** |
| **InstantDM** | $25〜 | Claude AI連携で返信文生成・日本語対応未確認 | ○ |
| **Spur AI** | $12〜/月 | AI Acquireプランで最安・Meta公式パートナー | ○ |

### Inrō推奨理由

- **フリープランで月100件まで自動化可能**（テスト十分）
- Pro €12.99/月で月500件以上の自動化が無制限
- コメント検知 → DM送信 → AI会話継続 → カレンダーURL送付まで一気通貫
- GDPR・Meta AIポリシー準拠で凍結リスクなし

## ②Threads自動化（投稿 + コメント自動返信）

### Threads APIの現状（2026年4月）

| 機能 | 対応状況 |
|------|----------|
| 投稿スケジュール | ○（Publishing API対応） |
| コメント読み取り | ○（`threads_manage_replies`権限） |
| コメントへの自動返信 | ○（APIで直接返信可能） |
| Webhookリアルタイム通知 | ○（コメント時に即座に通知） |
| コールドDM送信 | × （Threadsにはそもそもダイレクトメッセージ機能なし） |

### n8n + Claude連携の実装イメージ

```
[Threads投稿] → 誰かがコメント
        ↓
[Threads API Webhook] → n8nへリアルタイム通知
        ↓
[n8n] → コメント内容をClaude AIに送信
        ↓
[Claude] → 返信文生成（「詳細はLINEで→ [URL]」等）
        ↓
[n8n] → Threads APIで返信投稿
```

**実装コスト：**
- n8n セルフホスト：VPS $3/月（UpCloud等）
- Claude API：1,000往復で約$1〜2
- 開発工数：Claude AIを使えば2〜3日で構築可能

**注意事項：**
- Threads APIのApp Review（審査）が必要：申請から2〜4週間待ち
- `threads_manage_replies`権限の取得が前提
- Threadsはまだ日本国内でのビジネス活用事例が少ない（教育コストあり）

**実現可能性：○**（ただし即時ではなく2〜4週間後稼働）

### Threads活用のヒトモノチイキ最適戦略

- ThreadsはDMなし → 「コメント返信でLINEへ誘導」に特化
- 投稿テーマ：地域事業主の成功事例・インタビュー抜粋・「無料掲載受付中」告知
- 週3〜5投稿 + コメント自動返信で月20〜50件のLINE追加が見込める（仮説・未検証）

### 実装方法

**パターンA：LINE公式機能のみ（最小コスト）**
- AIチャットボット(β)機能を有効化（無料）
- キーワード応答でフロー設計
- Calendly無料プラン + IFTTT/Integrately でLINE通知連携
- **コスト：月5,000円（ライトプラン）のみ**
- **制限：自然言語処理の精度が低い・複雑なフロー組みにくい**

**パターンB：n8n + Claude + LINE API（推奨）**
- n8n セルフホスト（$3/月）
- Claude API（月1,000往復 ≒ $2〜5）
- LINE Messaging API webhook連携
- Calendly Free + n8n webhook
- **コスト：月1,000〜2,000円（n8n VPS）+ LINE月額5,000円 = 月約6,000〜7,000円**
- **品質：Claude AIによる自然な日本語対話・複雑なフロー対応可**

### n8n × LINE連携のセットアップ難易度

- 公式コミュニティノード（`n8n-nodes-line-messaging`等）が存在
- Claude AIにワークフロー設計を依頼すれば3〜5日で構築可能
- LINE Developers Consoleでチャンネル作成・Webhook URL設定が必要（難易度：低）

**実現可能性：◎**（最も確実・コスト効率が高い）

## 総括（Ryoの見解）

**SI Partnersの1日200通手動DMは、短期的には機能しているが長期的に持続不能なモデルだ。**

理由：
1. 非公式ツール依存の場合、Metaの検出強化でアカウント凍結リスクが増大中（2026年現在）
2. 200通→52件アポ（転換率26%）は表面的に高いが、コメント起点DM（転換率60〜80%）と比べると見劣りする
3. 人的工数（代理店費用）を自動化に置き換えることで、月コストを大幅削減しながら件数を維持・拡大できる

**推奨ポートフォリオ：**
- Instagram コメント→DM自動化（Inrō）+ LINE自動チャットボット（n8n×Claude）で2週間以内に稼働可能
- 月コスト：約15,000〜20,000円（現行SI Partners費用と比較して要確認だが、大幅に安い可能性が高い）
- 月+30件アポは2週間後から現実的に達成可能
