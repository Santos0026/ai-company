### 業種別仮説（未検証）

転換率が高い可能性が高い業種（Ryo仮説）：
1. 飲食・カフェ：SNS活用意識が高く、無料PRに敏感
2. 美容サロン：認知拡大ニーズが常にある
3. ハンドメイド作家：個人で発信しており掲載価値を理解しやすい
4. 教室・スクール：生徒募集のためPRを常に求めている

失注リスクが高い業種：
- 保険・金融・FP（競合するため）
- 大手チェーン（本社承認が必要）
- 医療・歯科（広告規制あり）

## 自動集客エンジン調査結果（2026-04-20）

### 重大な発見：SI Partnersのコールドは長期リスク
- Meta Graph API公式経由ではコールドDM（見知らぬユーザーへのDM）は送信不可能
- SI Partnersの手動200通/日は非公式ツール依存の可能性が高い → アカウント凍結リスクあり
- 代替として「コメント→DM自動化（合法）」が最有力

### 推奨自動化スタック（月15,000〜20,000円）

| ツール | 用途 | 月額 |
|--------|------|------|
| Inrō（フリー→Pro） | Instagram コメント→DM自動化 | 0〜€12.99 |
| n8n（VPS自己ホスト） | 全体のワークフロー自動化エンジン | $3〜 |
| LINE公式 ライトプラン | 友達追加→チャットボット→アポ確定 | 5,000円 |
| Claude API | 日本語チャットボット・返信生成 | 500〜1,000円 |
| Calendly（無料） | 日程調整・自動確定 | 0円 |
| Threads API（審査待ち） | 投稿コメント→LINE誘導 | 0円 |

### コメント→DM自動化のパフォーマンス指標
- 開封率：60〜80%（コールドDMの20〜30%の2〜4倍）
- コメントトリガーDM→返信率：60%（inroのデータ）
- 期待月間アポ：+30〜45件（2週間後から）
