## 午前の部（9時までに完了）

### STEP1：本日の投稿確認

Ryoのコンテンツカレンダーを確認：
- 本日投稿日か？（月・水・金）
- 投稿テーマは何か？
- 投稿テンプレートを担当者（チーム）に送る

投稿テンプレートの送り方：
```
【本日のThreads投稿です】

テーマ：○○（問いかけ型 / 事例型 / 募集型）

---
（本文をここに貼る）
---

投稿お願いします！#hitomono_chiiki.interview
```

### STEP2：昨日の反応を確認・記録

```
昨日の投稿反応：

## 随時（反応が来たら即座に動く）

### STEP3：コメント者の確認（コメントが来たら24時間以内）

確認項目：
- [ ] 個人事業主か？（プロフに業種・地域の記載があるか）
- [ ] 業種はヒトモノチイキのターゲットか？（整体・美容室・ネイル等）
- [ ] フォロワーが存在するアカウントか（10人以下は除外）
- [ ] 最近投稿しているか（3ヶ月以上投稿なしは除外）

OKなら → STEP4へ
NGなら → スキップ（記録だけする）

### STEP4：コメント返信文を担当者に提供

```
【コメント返信してください】

アカウント：@○○○
返信文：

「コメントありがとうございます😊
○○（業種）されてる方なんですね！
もし良ければ少し詳しくお話しできますか？
DM送っても大丈夫ですか？」
```

### STEP5：DM許可が取れたら → アポ打診文を提供

```
【DM送ってください】

アカウント：@○○○
DM文：

はじめまして、ヒトモノチイキの○○です😊

私たちは地域で活動されている方の想いをInstagramで無料発信するメディアです。

もし少しでもご興味あれば、
「興味あります」とだけ返信いただければ大丈夫です😊

## Sora の優先対応ルール

1. コメントが来たら **24時間以内** に返信文を用意する
2. DM許可が取れたら **当日中** にアポ打診文を用意する
3. アポ確定したら **即座** にKaiへ引き継ぐ
4. 個人事業主でない反応者（一般消費者）は丁寧に返信して終了
