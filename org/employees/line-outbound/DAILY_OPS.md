# Kai — 日次オペレーション手順書

**毎日この手順通りに動く。例外なし。**

---

## 午前の部（9時までに完了）

### STEP1：本日の送信リストを確定する

確認先：
- Luna（エキテン・Googleマップリスト）
- Hana（SNS発掘リスト・掲載投稿コメント欄リストも含む）
- Kento（ホットペッパー・Googleマップリスト）

チェック項目：
- [ ] 本日分の新規リストが届いているか
- [ ] 既送付リストとの重複を除外したか
- [ ] 合計200件以内になっているか（超える場合は翌日分へ）

### STEP2：送信文を確認する

現行パターン：
- **パターンA（現行）**：フォロワー1.7万・掲載100名超の実績入り
- **パターンB（テスト中）**：A/Bテスト比較用

本日使用パターンを記録する。

### STEP3：SI Partnersへ送信リストを共有

Googleスプレッドシートに本日分を追加し、以下のLINEを送る：

```
おはようございます！Kai（ヒトモノチイキ）です。
本日分のリストを追加しました。

本日：【○】件（エリア：東京○件・福岡○件・大阪○件）
使用パターン：A / B

よろしくお願いします！
[スプレッドシートURL]
```

## 午後の部（17時に確認）

### STEP4：当日の返信・アポ数を記録

SI Partnersから当日の結果を確認し、以下を記録：

```
【本日の結果】
送信数：     件
返信数：     件
アポ確定：   件
DM→返信率：  %
返信→アポ率： %
```

### STEP5：異常検知

直近5日の平均と比較して：
- 返信率が20%以上低下 → Jobsに即報告
- アポ転換率が20%以上低下 → 文面変更を検討・Jobsへ提案
