## 4. 代理店（保険FP）への納品推奨フォーマット

### 推奨CSVカラム構成
```
No | 店舗名 | 業種 | 地域（区） | 電話番号 | LINE/公式URL | Instagram | エキテンURL | スクリーニング | アポ担当者 | アポ日 | アポ結果 | 備考
```

### 追記フィールド（アポ管理用）
- **アポ担当者**：DM送信者・電話担当者名
- **初回接触日**：DM/電話を入れた日付
- **アポ結果**：未接触／送信済／連絡なし／アポ獲得／断り
- **面談日程**：FP面談の予定日
- **成約フラグ**：○（成約）／×（不成約）

### 月次KPI管理テンプレート
```
月次送信目標：200通
有効リスト目標：200件/月（○のみ）
目標アポ率：5〜10%（月10〜20件アポ）
目標成約率：30〜50%（月3〜10件FP面談成立）
```

## 5. 今後の収集戦略（スケールアップ方法）

### フェーズ1（現状：手動収集）
- エキテン各ページを業種×区ごとに巡回
- 1ページ40件 × 各業種8〜10ページ = 320〜400件/業種/区
- **優先区**：渋谷区・新宿区・豊島区・世田谷区・目黒区

### フェーズ2（自動化）
- エキテンURLパターン（`/gXXXX/aXXXXX/pN/`）を活用したCSVダウンロードツール
- Googleスプレッドシート＋GASでの自動整形・重複除去

### 優先DM送信順
1. **ネイルサロン・エステ（渋谷区・豊島区）** → 女性オーナー多、Instagram連携意識高
2. **料理教室・ヨガ（世田谷区）** → コミュニティ密着型、SNS発信に前向き
3. **美容室（新宿区・渋谷区）** → 件数が最多、大手除外後の小規模店を優先
4. **カフェ（全区）** → Instagram発信と親和性高い

## 6. エキテン掲載情報の注意点

- **電話番号**：未掲載店舗は公式HP・Googleビジネス・Instagramで補完が必要
- **Instagram**：エキテン上に掲載されているケースは少ない。Instagramは別途検索を推奨
- **「店舗公式」バッジ**：オーナーが能動的に登録・管理している証。接触しやすいオーナー傾向
- **LINE Pay対応**：LINE活用に抵抗が少ないことを示す重要シグナル
- **口コミ件数**：少ない店舗ほどSNS集客に課題感あり → Instagramメディア掲載の提案訴求ポイントになる
