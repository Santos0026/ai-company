# BO自動化システム セットアップガイド

## 必要なもの（全部無料〜月数千円）

| ツール | 用途 | コスト |
|--------|------|--------|
| Google Apps Script | スプシ監視・Webhook送信 | 無料 |
| n8n（クラウド or VPS） | ワークフロー自動化 | 月500〜2,000円 |
| LINE Messaging API | メッセージ自動送信 | 既存プラン内 |
| Claude API | 返信内容の分類・自動回答 | 月500〜1,000円 |
| Vercel / Netlify | BOダッシュボードのホスティング | 無料 |

---

## セットアップ手順

### STEP 1：Santos が用意するもの（5点）

1. **LINEチャンネルアクセストークン**
   - LINE Developers → チャンネル → 「Messaging API設定」
   - 「チャンネルアクセストークン（長期）」を発行してJobsに共有

2. **タスカルのスプレッドシートの列構成**
   - 「マッチング待ち」シートの各列の意味を教えてください
   - 特に：インタビュー実施日が入っている列はどれですか？

3. **n8nサーバーの設定**
   - Jobsがセットアップします（Santos はアカウント作成だけ）
   - https://n8n.io でアカウント作成 → 無料クラウドプランを選択

4. **Claude APIキー**
   - https://console.anthropic.com でアカウント作成
   - APIキーを発行してJobsに共有

5. **アンケートフォームのURL**
   - 電話後に送るアンケートフォームの固定URLを教えてください

---

### STEP 2：Jobs が実装するもの

1. Google Apps Scriptをスプレッドシートに設置
2. n8nワークフロー（4営業日計算・Webhook受信・LINE送信）を構築
3. BOダッシュボードをVercelにデプロイ
4. Claude AIと統合（返信分類・自動回答）
5. 本番テスト・調整

## 完成後のBOの1日

```
09:00 LINEに「今日の後確リスト」が自動で届く
       ↓
09:00〜 リストの人に電話（現行通り）
       ↓
電話後  ダッシュボードを開く → 結果をワンタップ入力
         ・✅ 日程確定 → 日時を入力して送信ボタン
         ・📋 未定   → 送信ボタン
         ・❌ FP未連絡→ 送信ボタン
         ・⚠️ 名前不一致→ 送信ボタン
       ↓
       LINEが自動送信される（5秒以内）
       スプシが自動更新される
       ↓
【自動】4日後・8日後にリマインダーが自動送信
【自動】お客様の「掲載いつ？」質問にClaude AIが自動回答
```

**BOが電話以外でやることは「ワンタップ入力」だけになります。**
