# n8n WF① 設定変更手順

## やること
Google Sheetsノードの読み取り対象を「顧客リスト」→「本日の後確」に変更する

---

## 手順

### 1. WF①を開く
n8nダッシュボード → 「BO①：毎朝9時 後確リスト通知」

### 2. Google Sheetsノードを変更
現在の設定：
- Sheet名: `顧客リスト`

変更後：
- Sheet名: `本日の後確`
- Filters: なし（GASが既にフィルター済み）

### 3. その後のノードのフィールド名を確認
「本日の後確」シートの列：
- A列: 顧客ID
- B列: 顧客名
- C列: フリガナ
- D列: インタビュー日
- E列: インタビュー時間
- F列: 処理日

LINEメッセージ生成ノードでこれらの列名を参照してください。

---

## 動作フロー（完成後）

```
毎朝8:55 → GAS実行
  CMS「顧客リスト」P列で4営業日後の顧客を抽出
  FPスプシR列が白色の顧客だけ絞り込む
  → CMS「本日の後確」シートに書き込み

毎朝9:00 → n8n WF①実行
  CMS「本日の後確」シートを読む
  → BOのLINEに通知
```
