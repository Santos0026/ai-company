## 全体フロー図

```
【スプレッドシート監視（n8n・常時）】

インタビュー完了日が記録される
        ↓ 4営業日後を自動計算
        ↓ n8nがタイマーセット

─────────────────────────────────
Day 4（4営業日後）：LINE自動送信①
─────────────────────────────────

【FPの進捗ステータスを確認】

ステータスA：日程確定済み
    → LINE①-A（日程確認バージョン）

ステータスB：マッチング済み・連絡待ち
    → LINE①-B（連絡待ちバージョン）

ステータスC：FP未連絡
    → LINE①-C（未連絡バージョン）

ステータスD：認識ズレ・未対応
    → LINE①-D（趣旨説明バージョン）

        ↓ お客様から返信が来る
    ┌────────────────────────────────┐
    │         Claude AIが返信を分類         │
    └────────────────────────────────┘
        ↓                          ↓
  日時を教えてくれた           まだ / 返信なし
        ↓                          ↓
  スプシ自動更新           4日後リマインダーをセット
  （日時・氏名記入）
        ↓
  LINE送信：予約確認＋注意事項
        ↓
  アンケートフォームを自動送信

─────────────────────────────────
Day 8（返信なしの場合）：LINE自動送信②
─────────────────────────────────
「その後いかがでしょうか？」テンプレ

─────────────────────────────────
Day 12（さらに返信なしの場合）：LINE自動送信③
─────────────────────────────────
「キャンセルポリシー」テンプレ

─────────────────────────────────
随時：お客様からの質問に自動回答
─────────────────────────────────
「掲載いつ？」→ 進捗確認して自動返信
```

## LINEメッセージテンプレート（自動送信版）

### LINE①-B（マッチング済み・連絡待ち）

```
お世話になっております！
ヒトモノチイキ事務局でございます😊

先日は無料インタビューのお時間をいただき
ありがとうございました！

本日は掲載記事の進捗と今後の流れについて
ご確認のご連絡をさせていただきました。

〇〇様の掲載記事は現在文面を作成中でございます。
この後LINEにて壁打ちのご連絡をさせていただき、
最終ゴーサイン後に掲載となりますので
引き続きよろしくお願いいたします😊

またご登録いただいた実施アンケートについてですが、
その後担当者よりメールまたはお電話にて
日程調整のご連絡はいただけましたでしょうか？

✅ 日程が決まった方は日時をLINEで教えてください！
📋 まだ決まっていない方は「未定」とご返信ください。
```

### LINE①-C（FP未連絡の場合）

```

```

お申し込みから一定期間ご連絡がない場合、
弊社のキャンセルポリシーに該当となる

お願い申し上げます。

ご不明な点がございましたら
何でもご連絡ください😊
```

```

## 技術構成

```
Google Sheets（FP進捗入力・顧客管理）
        ↓ n8n（変更検知・タイマー管理）
        ↓
Claude AI（返信内容の分類・返信文生成）
        ↓
LINE Messaging API（メッセージ送受信）
        ↓
Google Sheets（日時・ステータスの自動更新）
```

**月コスト：**
- n8n：月500〜1,000円（VPS）
- Claude API：月1,000〜2,000円（メッセージ量による）
- LINE公式：既存プラン内
- **合計：月1,500〜3,000円**

## BO担当者（人間）がやること（残り業務）

自動化後にBOが対応する必要があるのは：
1. **FPからの電話報告を受けてスプシを更新**（FPが電話の場合のみ）
2. **Claude AIが対応できない複雑な質問への対応**（月5件以下の想定）
3. **認識ズレがひどい場合の電話フォロー**（AIに切り替えが難しいケース）

それ以外は全自動。BOの工数を**80〜90%削減**できます。
