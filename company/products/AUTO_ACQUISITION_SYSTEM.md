# 自動集客エンジン 実装計画書

**作成**: 2026-04-17 by Jobs
**目標**: SI Partnersに頼らず月+30〜45件のアポを自動取得する

---

## システム全体図

```
【流入チャネル】

Instagram投稿（コメント誘導型）
    ↓ コメント「知りたい」
    Inrø（Meta認証・自動DM）
    ↓ 自動DM「LINE友達追加はこちら」
    ↓

Threads投稿（月・水・金）
    ↓ コメント発生
    Threads API Webhook
    ↓ n8n → Claude AI 自動返信
    ↓ 「詳細はLINEで」
    ↓

〈既存〉アウトバウンドDM（SI Partners・Kai管理）
    ↓ 興味あり返信
    ↓

            ▼ 全チャネルがここに集約
    ─────────────────────────────
    │  LINEチャットボット（核心）         │
    │  n8n × Claude AI × Calendly   │
    ─────────────────────────────
            ↓ 3問ヒアリング
            ↓ 条件合致 → 日程調整自動化
            ↓
         アポ確定🎯
```

## Phase 1：今週中に実装（コスト：ほぼ0円）

### 1-A. Inrøでコメント→DM自動化（1日で完了）

**手順：**
1. https://inro.social でフリープラン登録
2. Instagramビジネスアカウントと連携
3. キーワード設定：「知りたい」「興味あり」「詳しく」
4. 自動DM文を設定：

```
ありがとうございます😊

ヒトモノチイキの担当者です！
詳細はLINEでご案内させていただきます。

▼ LINE友達追加はこちら
[公式LINEのURL]

追加後すぐにご案内します！
```

5. コメント誘導投稿を作成：

```
「無料でInstagramに掲載してみませんか？」

整体院・美容室・ネイルサロンなど
地域で活動されている方を取材・掲載しています。

詳細が気になる方は
コメント欄に「知りたい」と書いてください！
すぐにDMでご案内します😊

#整体院 #美容室 #ネイルサロン #個人事業主 #無料掲載
```

**期待効果：** 月+15〜25件の LINE友達追加

---

### 1-B. LINEチャットボット（3〜5日で完了）

**必要なもの：**
- LINE公式アカウント ライトプラン（月5,000円）
- n8n（セルフホスト or クラウド）
- Claude API キー
- Calendly（フリープラン）

**チャットボットのフロー：**

```
友達追加
↓
[Claude] 自動ウェルカムメッセージ：
「ヒトモノチイキです😊
いくつかご質問させてください！」

Q1:「Instagramアカウントはお持ちですか？」
→ はい: Q2へ
→ いいえ: 「ありがとうございます。Instagramがない方は
          現在対応が難しい状況です。申し訳ありません」→ 終了

Q2:「どんなお仕事をされていますか？
     （整体院・美容室・ネイル・ヨガ等）」
→ 回答を記録

Q3:「一つだけ確認させてください。
     掲載にあたって保険の無料相談（60分ほど）に
     ご協力いただくことが条件となっています。
     こちらは大丈夫でしょうか？」
→ はい: 日程調整へ
→ いいえ/保険関係: 「承知しました。ご確認ありがとうございます」→ 終了

↓ 全条件クリア

「ありがとうございます！🎉
インタビューの日程を選んでください😊

▼ 日程調整はこちら
[Calendlyリンク]

選んでいただいた後、詳細をLINEでお送りします！」

↓ Calendlyで日程確定

[自動] 確認メッセージ：
「○月○日(○) ○時にインタビューですね！
ZoomのリンクをLINEでお送りします。
当日よろしくお願いします😊」
```

**技術構成（n8n）：**
```
LINE Webhook → n8n
↓
Claude API（会話状態管理・返信文生成）
↓
LINE Messaging API（返信送信）
↓
Calendly API（日程確認）
↓
Googleカレンダー（自動登録）
```
